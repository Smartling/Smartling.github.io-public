--- layout: page title: API Sandbox ---

The Smartling Sandbox API allows you to test API connection and parameters. The Sandbox API is available to registered API users but does not save or return data based on your actual account or project, and the Smartling Dashboard will not reflect data submitted to the sandbox.

  • File uploads will cache files without affecting production files and will not extract content for translation.
  • File downloads will always return the referenced files translated completely in "Reversi" (characters of each word returned in reverse order).
  • File statistics services will return mock statistics and metrics for files you've previously uploaded into the sandbox.
  • Even if you specify a callback on the file upload, it will not occur when using the Sandbox API

The purpose of the Sandbox API is to test API calls, so Smartling does not process data from the file. Uploaded files will not appear in the dashboard with their associated strings.

The Sandbox API is accessible at https://sandbox-api.smartling.com/v1 and all service calls have no affect on production data.

All URLs within the Sandbox API have the following base: 

https://sandbox-api.smartling.com/v1

To get started with the Sandbox API:

  1. Setup an account with Smartling, from: http://www.smartling.com/signup
  2. Obtain an API key within your Smartling dashboard: https://dashboard.smartling.com/settings/api
  3. Review the Smartling Translation API documentation at: https://docs.smartling.com
  4. Use the Sandbox API URL to execute API requests: https://sandbox-api.smartling.com/v1

Sandbox data may be cleared from time to time, so don't leave anything in the sandbox that you aren't willing to lose.
